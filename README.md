# DIP-Object-Classification

A Digital Image Processing project focused on image enhancement, noise filtering, segmentation, morphological processing, feature extraction, PCA analysis, and object classification.

## Project Features
- Noise Injection & Restoration
- Spatial and Frequency Domain Filtering
- Histogram Equalization & Gamma Transformation
- Thresholding & Segmentation
- Morphological Operations
  - Erosion
  - Dilation
  - Opening
  - Closing
  - Boundary Extraction
- Feature Extraction
  - Area
  - Perimeter
  - Circularity
- PCA Mapping & Object Classification
- MSE & PSNR Evaluation

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn
- Gradio
- Matplotlib

## Project Workflow
1. Image Enhancement & Restoration
2. Segmentation & Morphology
3. Feature Extraction
4. PCA Analysis
5. Object Classification

## Interface Preview
Interactive framework for applying image processing techniques and analyzing object features dynamically.
